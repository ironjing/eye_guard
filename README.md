# Mac Eye Guard

一个 macOS 护眼小程序。别名：眼睛护士/眼睛卫视/护眼计时程序

默认规则：
- 工作 20 分钟
- 休息 20 秒
- 每隔 3 次休息进入 5 分钟长休

休息期间会在所有屏幕显示黑色遮罩，展示倒计时和 `已休息好` 按钮。

## 启动
下载：MacEyeGuard.zip   解压缩后，直接双击打开即可。

## 可配置项
在菜单栏中打开 `设置`：
- 工作时长（分钟）
- 短休时长（秒）
- 长休时长（分钟）
- 每隔几次进入长休
